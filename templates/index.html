How to use

1 Choose a file to upload

This file may be projects done by students or a topic and abstract to check for plagiarism. In .docx format.

2 Click "Upload File"

This will upload the selected file into the database.

3 Click "Check Plagiarism"

This will display the files have have similar contents in the "Plagiarism Result" table.

  • "Newly Uploaded"

    • This file has similar contents with "Existed files".

  • "Existed files"

    • This file has similar contents with "Newly Uploaded".

  • "Plagiarism result"

    • This shows the amount of contents that might be similar (in percentage).
